Participant dropout in a longitudinal study is a potential source of bias. However, Bell (2013) shows differential dropout does not always bias results.

library(tidyverse)

The knitr package controls the height and width of the figure with options fig.height=5 and fig.width=7 where 5 and 7 are default values measure in inches. If you have a lot of content

base_dat <- expand.grid(id = 1:200, time = 0:2) %>%
  mutate(
    group = factor(if_else(id <= 100, 0, 1)),
    time = factor(time, levels = c(0, 1, 2))
  ) %>%
  arrange(id, time)

beta1 <- c(50, -3.5, -7.0, 1, (1 / 3.5))

linear_decline <- base_dat %>%
  mutate(y = beta1[1] + 
           (beta1[2] * (time == 1)) * if_else(group == 0, beta1[4], beta1[5]) + 
           (beta1[3] * (time == 2)) * if_else(group == 0, beta1[4], beta1[5]))

linear_decline %>% ggplot(aes(x = as.numeric(time) - 1, y = y)) +
  geom_line(aes(linetype = group)) +
  scale_y_continuous(limits = c(40, 52), n.breaks = 7)

beta2 <- c(50, -4, -3, 1, (7 / 4))

temporary_change <- base_dat %>%
  mutate(y = beta1[1] + 
           (beta2[2] * (time == 1)) + (if_else(group == 1, -3, 0) * (time == 1)) + 
           (beta2[3] * (time == 2)) + (if_else(group == 1, 5, 0) * (time == 2)))
 
temporary_change %>% ggplot(aes(x = as.numeric(time) - 1, y = y)) +
  geom_line(aes(linetype = group)) +
  scale_y_continuous(limits = c(40, 52), n.breaks = 7)

References

Bell, M. L., Kenward, M. G., Fairclough, D. L., & Horton, N. J. (2013). Differential dropout and bias in randomised controlled trials: when it matters and when it may not. BMJ (Clinical research ed.), 346, e8668. https://doi.org/10.1136/bmj.e8668. html.
